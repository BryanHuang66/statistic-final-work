本文件夹一共包含6个jupyter notebook文件，1个r文件以及1个matlab文件。

为了便于写清楚各个方法、数据之间的区别，因此将python相关内容分成6个文件，其中6个jupyter notebook文件对应了6个数据表。由于r语言和matlab内容比较少，所以仅仅使用了一个文件，包含了所使用的方法。


## 安装python依赖

